SSN Incident Management Application

Please use this portal to report an incident for any issues you face or observe. Examples are System not working, Internet / Wi-Fi issues,UPS,Electrical / Plumbing, RO Water, Intercom, AC, Transport, Attendance,Cafeteria etc)

The process is simple.

1) Login to the tool (Log In) using Digital ID Credentials.

2) Click “Open a New Ticket”.

2) Select appropriate Category from Help Drop down.

3) Give a description of the issue.

4) Click on “Create Ticket”.

5) Share your mobile number (optional) to speed up the resolution process.
